# EcoAuto

Brand site for an EV automotive service. Built as a single‑page HTML landing page with hero video, service grid, gallery, and contact funnel.

## Design

- **Accent:** electric green (`#00ff88`) on a soft greyscale gradient body
- **Type:** Inter for UI, SK Greenland for display
- **Hero:** autoplay video loops (`vid1.mp4`, `vid2.mp4`)
- **Smooth scroll** with anchor‑linked nav

## Stack

Plain HTML and CSS. Native video, custom design tokens declared as CSS variables. No build step. Just open `index.html`.

## Structure

```
index.html         page
vid1.mp4, vid2.mp4 hero video loops
1.jpg ... 12.jpg   gallery imagery
logo.png, car.png  brand assets
```

## Contact CTAs

WhatsApp and Instagram icons link out to the client's accounts.
